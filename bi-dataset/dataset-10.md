## 설명

이 데이터셋은 VCworks의 [조합 정보, 조합별 운용인력 정보, 투자의무 정보]를 결합하여 모태펀드나 성장금융 출자사업제안서 작성에 필요한 조합 운용 현황을 확인할 수 있도록 만든 데이터셋입니다. [약정총액, 납입총액, 투자총액, 회수총액, 투자의무 목적과 달성 여부]를 조합별로 제공하므로 대표펀드매니저나 핵심운용인력 기준의 조합 운용실적 표를 구성하는 데 적합합니다.

데이터는 분기말일을 기준으로 한 누적 스냅샷이며, 각 `기준일자` 값은 해당 시점까지 전체 기간을 누적한 현황입니다. 누적 자체는 전체 기간을 기준으로 합산되지만, 제공되는 스냅샷이 최근 5개년의 분기말 시점으로 한정될 뿐입니다. 같은 조합이 분기말 기준일자마다 반복되어 쌓여 있으므로, 반드시 **단일 `기준일자`로 필터링하거나 `기준일자`를 분석 기준(축)에 포함**해야 합니다. 그렇지 않으면 여러 시점의 누적값이 함께 더해져 금액이 크게 부풀려집니다.

금액 값들은 각 기준일자 시점까지 이미 누적된 값이므로 따로 누적 합산을 하시면 안됩니다. 또한 같은 기준일자 안에서도 한 조합에 운용인력이나 투자의무가 여러 개면 같은 조합 금액이 여러 행에 반복되어 나옵니다. 그래서 그대로 합치면 금액이 중복으로 부풀려지니, 조합별 금액을 볼 때는 조합 기준으로 보고(중복 제거), 운용인력별로 볼 때는 운용인력 컬럼을 기준에 넣어서 봐야 합니다.

## 제공 컬럼

| 컬럼명 | 설명 |
| --- | --- |
| 기준일자 | 해당 데이터가 집계·산출된 기준 날짜입니다. |
| 조합명 | 투자조합의 공식 명칭입니다. |
| 조합_고유번호 | 투자조합의 고유 식별번호(사업자등록번호)입니다. |
| 조합_운용_상태 | 투자조합의 현재 운용 상태입니다. |
| 조합_유형 | 투자조합의 법적 유형입니다. (예: 벤처투자조합, PEF 등) |
| 약정총액 | 투자조합의 약정총액입니다. |
| 납입총액 | 조합원으로부터 실제로 납입된 총 금액입니다. |
| 투자총액 | 현재까지 집행된 총 투자 금액입니다. |
| 회수총액 | 회수된 총 금액입니다. |
| 존속기간_시작일 | 투자조합의 존속기간 시작일(결성일)입니다. |
| 투자기간_종료일 | 투자조합의 투자기간 종료(예정)일입니다. |
| 해산일자 | 투자조합의 해산(예정)일입니다. |
| 청산일자 | 투자조합의 청산(예정)일입니다. |
| 기준수익률 | 투자조합의 기준수익률입니다. |
| 운용인력_성명 | 해당 투자조합의 운용인력 성명입니다. |
| 운용인력_구분 | "운용인력의 구분입니다. (예: 대표펀드매니저, 핵심운용인력 등)" |
| 운용인력_시작일 | 운용인력이 해당 조합에 배정되어 업무를 시작한 날짜입니다. |
| 운용인력_종료일 | 운용인력이 해당 조합에서 배정 업무를 종료한 날짜입니다. |
| 운용인력_재직여부| 현재 운용인력이 재직 중인지 여부입니다. |
| 투자의무_목적명 | 규약 또는 계약에 따른 투자 의무의 목적을 나타내는 명칭입니다. |
| 투자의무_조건 | 투자 의무 이행과 관련된 조건 또는 요건입니다. |
| 투자의무_부여LP명 | 투자 의무를 부여한 특정 LP의 명칭입니다. |
| 현재_달성금액 | 현재 시점까지 누적 달성한 금액입니다. |
| 의무_한도금액 | 투자 의무액 또는 한도액을 나타냅니다. |
| 의무_달성여부 | 해당 투자 의무가 달성되었는지 여부입니다. |
