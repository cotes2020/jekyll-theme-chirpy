## 설명

이 데이터셋은 투자 건별 수익률을 월별로 확인하기 위한 데이터셋입니다. VCworks의 "수익률 조회(Gross)" 화면에서 다루는 포트폴리오별 수익률 시계열을 BI 분석용으로 정리한 것으로, [기준일자, 조합 정보, 포트폴리오명, 포트폴리오 국가, 기업규모, 납입일자, 투자유형, 최초투자조건id, 원금평가·가치평가 기준 GROSS IRR과 Multiple, 투자금액, 회수금액, 미회수 원금, 미회수 가치평가 금액, 평가기준]을 제공합니다.

[포트폴리오, 투자조건] 단위로 어떤 투자 건의 수익률이 어떻게 변하는지, 원금평가와 가치평가 기준의 성과가 어떻게 다른지 확인할 때 적합합니다. 데이터는 월말일을 기준으로 한 누적 스냅샷이며, 각 `기준일자` 값은 해당 시점까지 전체 기간을 누적한 현황입니다. 누적 자체는 전체 기간을 기준으로 합산되지만, 제공되는 스냅샷이 최근 5개년의 월말 시점으로 한정될 뿐입니다. 같은 투자 건이 월말 기준일자마다 반복되어 쌓여 있으므로 반드시 **단일 `기준일자`로 필터링하거나 `기준일자`를 분석 기준(축)에 포함**해야 합니다. 그렇지 않으면 여러 시점의 값이 섞여 잘못 해석되며, 특히 IRR·Multiple 같은 수익률 지표는 월별 값을 단순 합산·평균하면 안 됩니다. 투자 건 단위의 Gross 성과를 보는 데이터셋이므로 [조합 전체의 Net 성과, 운용사 전체 수익률]과는 분석 목적을 구분해야 합니다.

## 제공 컬럼

| 컬럼명 | 설명 |
| --- | --- |
| 기준일자 | 해당 데이터가 집계·산출된 기준 날짜입니다. 월 말일로만 제공됩니다. |
| 조합명 | 투자조합의 공식 명칭입니다. |
| 조합_약칭명 | 투자조합의 약칭입니다. |
| 조합_유형 | 투자조합의 유형(예: 벤처투자조합, PEF 등)입니다. |
| 조합_운용_상태 | 투자조합의 현재 운용 상태입니다. |
| 존속기간_시작일 | 투자조합 존속기간이 시작된 날짜(결성일)입니다. |
| 포트폴리오명 | 포트폴리오의 명칭입니다. |
| 포트폴리오_국가 | 포트폴리오가 소재한 국가입니다. |
| 기업규모 | 포트폴리오의 규모 구분(예: 대기업, 중소기업 등)입니다. |
| 납입일자 | 투자금이 실제 납입된 날짜입니다. |
| 투자유형 | 해당 거래의 투자 유형입니다. (예: 주식, 채권 등) |
| 세부투자유형 | 투자 유형의 세부 분류입니다. (예: 보통주, 우선주, CB 등) |
| 신주_구주 | 해당 거래가 신주인지 구주인지를 표시합니다. |
| 최초투자조건id | 해당 투자와 연결된 최초 투자 조건의 고유 식별값입니다. |
| 원금평가_GROSS_IRR | 원금 기준으로 계산한 GROSS 내부수익률(IRR)입니다. |
| 가치평가_GROSS_IRR | 가치평가 기준으로 계산한 GROSS 내부수익률(IRR)입니다. |
| 원금평가_GROSS_Multiple | 원금 기준으로 산출한 GROSS 투자배수(Multiple)입니다. |
| 가치평가_GROSS_Multiple | 가치평가 기준으로 산출한 GROSS 투자배수(Multiple)입니다. |
| 투자금액 | 해당 투자건의 투자 금액입니다. |
| 회수금액 | 해당 투자건의 회수 금액입니다. |
| 미회수_원금 | 해당 투자건의 아직 회수되지 않은 원금 잔액입니다. |
| 미회수_가치평가_금액 | 해당 투자건의 회수되지 않은 투자자산의 현재 가치평가 금액입니다. |
| 평가기준 | 투자자산 가치평가 시 적용한 기준입니다. |
