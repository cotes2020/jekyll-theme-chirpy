## 설명

이 데이터셋은 투자 건별 가치평가 내역에 그 투자를 담당한 심사역과 기여율을 더해, 심사역 관점에서 투자 성과와 평가가치를 살펴볼 수 있도록 만든 데이터셋입니다. 투자 건별로 [기준일자, 조합, 포트폴리오, 투자유형, 투자금액, 회수원금·회수손익·회수총액, 투자잔액, 평가가치] 정보에 더해 [담당 심사역과 역할, 발굴·심사·사후관리 기여율, 총기여율]과 이 기여율을 반영한 금액을 함께 제공합니다. 심사역별로 어떤 기업에 투자해 얼마의 평가가치와 성과를 냈는지, 기여율을 반영하면 성과가 어떻게 달라지는지 분석할 때 활용할 수 있습니다.

데이터는 분기말일을 기준으로 한 누적 스냅샷이며, 각 `기준일자` 값은 해당 시점까지 전체 기간을 누적한 현황입니다. 누적 자체는 전체 기간을 기준으로 합산되지만, 제공되는 스냅샷이 최근 5개년의 분기말 시점으로 한정될 뿐입니다. 같은 투자 건이 분기말 기준일자마다 반복되어 쌓여 있으므로, 반드시 **단일 `기준일자`로 필터링하거나 `기준일자`를 분석 기준(축)에 포함**해야 합니다. 그렇지 않으면 여러 시점의 값이 함께 더해져 금액이 크게 부풀려집니다.

또한 하나의 투자 건을 여러 심사역이 함께 담당할 수 있어, 같은 기준일자 안에서도 투자 건 1건이 심사역 수만큼 여러 줄로 나뉘어 들어갑니다. 이 상태에서 투자금액이나 평가가치를 그대로 더하면 같은 금액이 중복으로 계산되므로, 심사역별 성과를 볼 때에는 반드시 심사역을 분류 기준이나 필터에 넣어야 합니다. 심사역과 무관하게 투자 건 자체의 가치평가 현황만 보고 싶을 때에는 "투자건 가치평가" 데이터셋을 사용하는 것이 좋습니다.

## 제공 컬럼

| 컬럼명 | 설명 |
| --- | --- |
| 기준일자 | 해당 데이터가 집계·산출된 기준 날짜입니다. |
| 포트폴리오명 | 포트폴리오의 명칭입니다. |
| 조합명 | 투자조합의 공식 명칭입니다. |
| 최초_투자유형 | 최초 투자 시점의 투자 유형입니다. (예: 보통주, 우선주, CB 등) |
| 현재_투자유형 | 현재 시점에서의 투자 유형입니다. (예: 보통주, 우선주, CB 등) |
| 신주_구주 | 해당 거래가 신주인지 구주인지를 표시합니다. |
| 투자일 | 해당 투자가 집행된 날짜(납입일자)입니다. |
| 투자라운드 | 해당 투자 라운드(Stage)입니다. |
| 투자전_기업가치 | 해당 투자 직전의 기업 가치입니다. |
| 투자후_기업가치 | 해당 투자 직후의 기업 가치입니다. |
| 투자금액 | 해당 투자에 집행된 금액입니다. |
| 회수원금 | 회수를 통해 회수된 금액 중 원금에 해당하는 부분입니다. |
| 회수손익 | 회수를 통해 발생한 손익(수익 또는 손실) 금액입니다. |
| 회수총액 | 원금 회수금액과 회수손익 금액의 합계입니다. |
| 잔액_단가 | 남아 있는 보유 주식에 대한 평균 단가입니다. |
| 보유주식수 | 현재 보유하고 있는 주식의 수량입니다. |
| 총발행주식수 | 포트폴리오의 총 발행 주식수입니다. (보통주 + 우선주) |
| 투자잔액 | 현재까지 회수되지 않고 남아 있는 투자 원금 잔액입니다. |
| 평가_기준일 | 해당 평가 가치가 산출된 기준 날짜입니다. |
| 기준일_평가_단가 | 기준일 기준 주식 평가 단가입니다. |
| 평가_가치 | 현재 기준의 투자자산 평가 금액입니다. |
| 평가_주체 | 투자자산에 대한 평가를 수행한 주체입니다. |
| 가치_산출_근거 | 평가 금액을 산출할 때 사용된 근거 또는 방법론입니다. |
| 주요제품명 | 포트폴리오의 주력으로 생산·판매하는 주요 제품명입니다. |
| 비즈니스분야 | 포트폴리오의 비즈니스 분야입니다. |
| 상위비즈니스분야 | 포트폴리오의 상위 비즈니스 분야입니다. |
| 국내_해외 | 포트폴리오의 국내/해외 구분입니다. |
| 투자조건id | 해당 투자와 연결된 투자 조건의 고유 식별값입니다. |
| 최초투자조건id | 해당 투자와 연결된 최초 투자 조건의 고유 식별값입니다. (심사역 정보를 연결하는 기준 값) |
| 조합id | 해당 투자 건이 속한 투자조합의 고유 식별값입니다. |
| 회수_평가가치_합계금액 | 해당 투자 건의 회수총액과 평가가치를 합산한 금액입니다. |

#### 심사역 컬럼

| 컬럼명 | 설명 |
| --- | --- |
| 심사역명 | 해당 투자조건에 연결된 담당 심사역명입니다. |
| 심사역_구분 | 해당 심사역의 조합 운용인력 구분입니다. (조합 운용인력인 경우) |
| 심사역_시작일 | 해당 심사역이 조합 운용인력으로 배정된 시작일입니다. (조합 운용인력인 경우) |
| 심사역_종료일 | 해당 심사역이 조합 운용인력으로 배정된 종료일입니다. (조합 운용인력인 경우) |
| 발굴_기여율 | 담당심사역의 발굴 기여율과 조합 성과율을 곱한 값입니다. |
| 심사_기여율 | 담당심사역의 심사 기여율과 조합 성과율을 곱한 값입니다. |
| 사후관리_기여율 | 담당심사역의 사후관리 기여율과 조합 성과율을 곱한 값입니다. |
| 총기여율 | 담당심사역의 기여율 총계입니다. |

#### 기여율 적용 컬럼

| 컬럼명 | 설명 |
| --- | --- |
| 기여율_적용_투자금액 | 투자금액에 총기여율을 적용한 금액입니다. (총기여율 × 투자금액 ÷ 100) |

| 기여율_적용_회수손익 | 회수손익에 총기여율을 적용한 금액입니다. (총기여율 × 회수손익 ÷ 100) |
| 기여율_적용_회수총액 | 회수총액에 총기여율을 적용한 금액입니다. (총기여율 × 회수총액 ÷ 100) |
| 기여율_적용_투자잔액 | 투자잔액에 총기여율을 적용한 금액입니다. (총기여율 × 투자잔액 ÷ 100) |
| 기여율_적용_평가가치 | 평가_가치에 총기여율을 적용한 금액입니다. (총기여율 × 평가_가치 ÷ 100) |
| 기여율_적용_회수_평가가치_합계금액 | 회수_평가가치_합계금액에 총기여율을 적용한 금액입니다. (총기여율 × 회수_평가가치_합계금액 ÷ 100) |
