## 설명

이 데이터셋은 VCworks의 "가치 평가" 화면에서 관리되는 투자 건별 가치평가 내역을 BI 분석용으로 정리한 데이터셋입니다. 각 투자 건에 대해 [기준일자별 투자금액, 회수원금, 회수손익, 회수총액, 투자잔액, 보유주식수, 평가 단가, 평가 가치, 평가 주체, 가치 산출 근거] 등을 함께 제공하므로 특정 시점의 투자자산 평가 현황을 [포트폴리오, 조합, 투자유형, 섹터] 등의 기준으로 나누어 살펴볼 수 있습니다.

데이터는 분기말일을 기준으로 한 누적 스냅샷이며, 각 `기준일자` 값은 해당 시점까지 전체 기간을 누적한 현황입니다. 누적 자체는 전체 기간을 기준으로 합산되지만, 제공되는 스냅샷이 최근 5개년의 분기말 시점으로 한정될 뿐입니다. 같은 투자 건이 분기말 기준일자마다 반복되어 쌓여 있으므로, 반드시 **단일 `기준일자`로 필터링하거나 `기준일자`를 분석 기준(축)에 포함**해야 합니다. 그렇지 않으면 여러 시점의 값이 함께 더해져 금액이 크게 부풀려지고, 별도의 누적 계산을 다시 적용하면 값이 중복 해석됩니다. 단순 합계를 사용할 때에도 [조합, 포트폴리오] 같은 분석 기준을 함께 고려해야 합니다. [투자사별, 조합별, 섹터별] 평가 멀티플이나 기준일 간 평가가치 변동을 확인하는 대시보드에 활용하기 적합합니다.

## 제공 컬럼

| 컬럼명 | 설명 |
| --- | --- |
| 기준일자 | 해당 데이터가 집계·산출된 기준 날짜입니다. |
| 포트폴리오명 | 포트폴리오의 명칭입니다. |
| 조합명 | 투자조합의 공식 명칭입니다. |
| 최초_투자유형 | 최초 투자 시점의 투자 유형입니다. (예: 보통주, 우선주, CB 등) |
| 현재_투자유형 | 현재 시점에서의 투자 유형입니다. (예: 보통주, 우선주, CB 등) |
| 신주_구주 | 해당 거래가 신주인지 구주인지를 표시합니다. |
| 투자일 | 해당 투자가 집행된 날짜(납입일자)입니다. |
| 투자금액 | 해당 투자에 집행된 금액입니다. |
| 회수원금 | 회수를 통해 회수된 금액 중 원금에 해당하는 부분입니다. |
| 회수손익 | 회수를 통해 발생한 손익(수익 또는 손실) 금액입니다. |
| 회수총액 | 원금 회수금액과 회수손익 금액의 합계입니다. |
| 잔액_단가 | 남아 있는 보유 주식에 대한 평균 단가입니다. |
| 보유주식수 | 현재 보유하고 있는 주식의 수량입니다. |
| 총발행주식수 | 포트폴리오의 총 발행 주식수입니다. (보통주 + 우선주) |
| 투자잔액 | 현재까지 회수되지 않고 남아 있는 투자 원금 잔액입니다. |
| 평가_기준일 | 해당 평가 가치가 산출된 기준 날짜입니다. |
| 기준일_평가_단가 | 기준일 기준 주식 평가 단가입니다. |
| 평가_가치 | 현재 기준의 투자자산 평가 금액입니다. |
| 평가_주체 | 투자자산에 대한 평가를 수행한 주체(조직 또는 담당자)입니다. |
| 가치_산출_근거 | 평가 금액을 산출할 때 사용된 근거 또는 방법론입니다. |
| 주요제품명 | 포트폴리오의 주력으로 생산·판매하는 주요 제품명입니다. |
| 투자조건id | 해당 투자와 연결된 투자 조건의 고유 식별값입니다. |
| 최초투자조건id | 해당 투자와 연결된 최초 투자 조건의 고유 식별값입니다. |
| 조합id | 해당 투자 건이 속한 투자조합의 고유 식별값입니다. |
| 지분_현재가치 | 해당 투자 건의 회수총액과 평가 가치를 합산한 금액입니다. |
