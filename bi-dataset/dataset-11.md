## 설명

이 데이터셋은 운용인력 기준으로 투자 건 단위의 상세 투자 현황을 분석하기 위한 데이터셋입니다. VCworks의 [심사역·운용인력 정보, 투자조건, 거래내역, 포트폴리오 정보]를 결합하여 [운용인력 성명과 역할, 조합, 포트폴리오, 투자유형, 투자금액, 회수총액, 미회수원금, 평가가치, 회수여부, 기여율] 정보를 함께 제공합니다. 모태펀드·성장금융 제안서에서 운용인력별 투자 이력과 회수 성과를 설명하거나, 운용인력별 포트폴리오 구성과 투자기간을 비교할 때 활용할 수 있습니다.

데이터는 분기말일을 기준으로 한 누적 스냅샷이며, 각 `기준일자` 값은 해당 시점까지 전체 기간을 누적한 현황입니다. 누적 자체는 전체 기간을 기준으로 합산되지만, 제공되는 스냅샷이 최근 5개년의 분기말 시점으로 한정될 뿐입니다. 같은 투자 건이 분기말 기준일자마다 반복되어 쌓여 있으므로, 반드시 **단일 `기준일자`로 필터링하거나 `기준일자`를 분석 기준(축)에 포함**해야 합니다. 그렇지 않으면 여러 시점의 누적값이 함께 더해져 값이 크게 부풀려집니다.

또한 같은 기준일자 안에서도 하나의 투자 건이 여러 운용인력 또는 기여율 정보와 연결될 수 있습니다. 따라서 [투자금액, 회수총액, 평가가치] 같은 금액을 단순 합산하면 의도보다 크게 보일 수 있으므로 [운용인력, 조합, 포트폴리오, 투자조건] 기준을 함께 두고 분석해야 합니다. 운용인력별 성과가 아니라 투자 건 자체의 전체 거래 흐름이 필요하다면 "전체 거래내역" 데이터셋을 사용하시는 것이 좋습니다.

## 제공 컬럼

| 컬럼명 | 설명 |
| --- | --- |
| 기준일자 | 해당 데이터가 집계·산출된 기준 날짜입니다. |
| 운용인력_성명 | 해당 투자조합의 운용인력 성명입니다. |
| 조합명 | 투자조합의 공식 명칭입니다. |
| 포트폴리오명 | 포트폴리오의 명칭입니다. |
| 투자심사id | 해당 투자 건이 연결된 투자 심사 고유 식별값입니다. |
| 운용인력_구분 | 운용인력의 구분입니다. (예: 대표펀드매니저, 핵심운용인력 등) |
| 운용인력_시작일 | 운용인력이 해당 조합에 배정되어 업무를 시작한 날짜입니다. |
| 운용인력_종료일 | 운용인력이 해당 조합에서 배정 업무를 종료한 날짜입니다. |
| 발굴_기여율 | 담당심사역의 발굴 기여율과 조합 성과율을 곱한 값입니다. |
| 심사_기여율 | 담당심사역의 심사 기여율과 조합 성과율을 곱한 값입니다. |
| 사후관리_기여율 | 담당심사역의 사후관리 기여율과 조합 성과율을 곱한 값입니다. |
| 총기여율 | 담당심사역의 기여율 총계입니다. |
| 투자계정구분 | 본계정이면 고유계정, 그 외에는 조합계정으로 구분됩니다. |
| 조합_고유번호 | 투자조합의 고유 식별번호(사업자등록번호)입니다. |
| 조합_유형 | 투자조합의 법적 유형입니다. (예: 벤처투자조합, PEF 등) |
| 포트폴리오_법인등록번호 | 포트폴리오의 법인등록번호입니다. |
| 포트폴리오_사업자등록번호 | 포트폴리오의 사업자등록번호입니다. |
| 포트폴리오_법인유형 | 포트폴리오의 법인 유형입니다. (예: 일반법인, 해외, 외부조합) |
| 기업규모 | 포트폴리오의 기업규모입니다. (예: 대기업, 중소기업 등) |
| 포트폴리오_표준산업분류 | 포트폴리오의 표준산업분류 코드입니다. |
| 투자일 | 투자금이 실제 납입된 날짜입니다. |
| 투자유형 | 해당 투자 건의 투자유형입니다. (예: 보통주, 우선주, CB 등) |
| 처분구분 | 자산 처분의 구분입니다. (예: 국내상장, 해외상장, 환매, 감액 등) |
| 투자기간 | 투자일로부터 최종 회수일까지 기간입니다. (단위: 년) |
| 가치_산출_근거 | 해당 투자 건 평가가치의 가치 산출 근거입니다. |
| 상장여부 | 포트폴리오의 상장 여부입니다. |
| 창업자여부 | 창업자가 포트폴리오의 주요 경영진에 포함되는지 여부입니다. |
| 설립일 | 포트폴리오의 법인 설립일입니다. |
| 벤처기업여부 | 포트폴리오가 벤처기업에 속하면 true, 아니면 false입니다. |
| 투자조건id | 해당 투자 건이 연결된 투자 조건 고유 식별값입니다. |
| 최초투자조건id | 해당 투자 건이 연결된 최초 투자 조건 고유 식별값입니다. |
| 투자금액 | 해당 투자 건의 투자금액입니다. |
| 회수총액 | 해당 투자 건의 회수총액입니다. |
| 회수원금 | 해당 투자 건의 회수원금입니다. |
| 미회수원금 | 아직 회수되지 않은 원금 잔액입니다. |
| 평가가치 | 해당 투자 건의 현재 평가가치입니다. |
| 최종_회수일 | 해당 투자 건의 회수, 단주대금 거래 중 마지막 날짜입니다. |
| 회수여부 | 회수 현황에 따라 미회수, 부분회수, 회수완료로 표기됩니다. |
