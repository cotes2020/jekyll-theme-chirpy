## 설명

이 데이터셋은 조합별 수익률과 관련 금액을 월별로 제공하는 데이터셋입니다. [조합명, 조합 유형, 운용 상태, 존속기간, 약정총액] 같은 조합 기본 정보와 함께 기준일자별 [원금평가·가치평가 기준 GROSS IRR, NET IRR, Multiple, 투자금액, 회수금액, 미회수 원금, 미회수 가치평가 금액, 출자금액, 배분금액, 미배분금액, 투자 종목수]를 제공합니다.

조합 단위의 성과를 비교하거나 특정 조합의 수익률 추이를 확인할 때 사용하는 데이터셋입니다. 데이터는 월말일을 기준으로 한 누적 스냅샷이며, 각 `기준일자` 값은 해당 시점까지 전체 기간을 누적한 현황입니다. 누적 자체는 전체 기간을 기준으로 합산되지만, 제공되는 스냅샷이 최근 5개년의 월말 시점으로 한정될 뿐입니다. 같은 조합이 월말 기준일자마다 반복되어 쌓여 있으므로 반드시 **단일 `기준일자`로 필터링하거나 `기준일자`를 분석 기준(축)에 포함**해야 합니다. 그렇지 않으면 여러 시점의 값이 섞여 잘못 해석되며, 특히 IRR·Multiple 같은 수익률 지표는 월별 값을 단순 합산·평균하면 안 됩니다. Gross 지표는 [투자, 회수] 흐름, Net 지표는 [출자, 배분] 흐름에 더 가깝게 해석되므로, [조합별 투자 성과, LP 관점의 출자·배분 성과]를 구분해서 차트를 구성하는 것이 좋습니다.

## 제공 컬럼

| 컬럼명 | 설명 |
| --- | --- |
| 조합명 | 투자조합의 공식 명칭입니다. |
| 조합_약칭명 | 투자조합의 약칭입니다. |
| 조합_유형 | 투자조합의 유형(예: 벤처투자조합, PEF 등)입니다. |
| 조합_운용_상태 | 투자조합의 현재 운용 상태입니다. |
| 존속기간_시작일 | 투자조합의 존속기간 시작일(결성일)입니다. |
| 약정총액 | 투자조합의 약정총액입니다. |
| 기준일자 | 해당 데이터가 집계·산출된 기준 날짜입니다. 월 말일로만 제공됩니다. |
| 원금평가_GROSS_IRR | 원금 기준으로 계산한 GROSS 내부수익률(IRR)입니다. |
| 가치평가_GROSS_IRR | 가치평가 기준으로 계산한 GROSS 내부수익률(IRR)입니다. |
| 원금평가_GROSS_Multiple | 원금 기준으로 산출한 GROSS 투자배수(Multiple)입니다. |
| 가치평가_GROSS_Multiple | 가치평가 기준으로 산출한 GROSS 투자배수(Multiple)입니다. |
| 원금평가_NET_IRR | 원금 기준으로 계산한 NET 내부수익률(IRR)입니다. |
| 가치평가_NET_IRR | 가치평가 기준으로 계산한 NET 내부수익률(IRR)입니다. |
| 원금평가_NET_Multiple | 원금 기준으로 산출한 NET 투자배수(Multiple)입니다. |
| 가치평가_NET_Multiple | 가치평가 기준으로 산출한 NET 투자배수(Multiple)입니다. |
| 투자금액 | 해당 투자조합의 현재까지 집행된 총 투자 금액입니다. |
| 회수금액 | 해당 투자조합의 투자 회수를 통해 회수된 총 금액입니다. |
| 미회수_원금 | 아직 회수되지 않은 원금 잔액입니다. |
| 미회수_가치평가_금액 | 아직 회수되지 않은 투자에 대한 현재 가치평가 금액입니다. |
| 출자금액 | 조합원으로부터 실제로 납입된 총 출자금액입니다. |
| 배분원금 | 조합원에게 배분된 금액 중 원금에 해당하는 부분입니다. |
| 배분_수익_금액 | 조합원에게 배분된 금액 중 수익(이익)에 해당하는 금액입니다. |
| 배분_합계_금액 | 조합원에게 배분된 원금과 수익의 합계 금액입니다. |
| 미배분원금 | 조합에서 아직 조합원에게 배분하지 않은 원금 잔액입니다. |
| 미배분_가치평가_금액 | 조합에서 아직 배분하지 않은 투자자산의 가치평가 금액입니다. |
| 투자_종목수 | 해당 투자조합이 투자한 종목(포트폴리오)의 개수입니다. |
