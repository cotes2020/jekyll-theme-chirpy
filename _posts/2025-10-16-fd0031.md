조합원총회 상세 화면의 **기본정보 탭**에서는 총회의 주요 정보를 확인하고 총회 진행에 필요한 작업을 수행할 수 있습니다.  
총회의 진행 상태에 따라 **개최 안내, 수정, 마감일 연장, 결과 전송** 등의 기능을 사용할 수 있습니다.

---
## 설명  
### 총회 정보 확인하기  
  
기본정보 탭에 들어오면 먼저 **총회의 주요 정보와 진행 상태**를 확인할 수 있습니다.  
  
확인할 수 있는 주요 정보는 다음과 같습니다.  
  
- 총회 제목  
- 진행 상태  
- 조합명  
- 총회 개최일시  
- 응답 기간  
- 조합원 응답률  
  
응답률은 총회 진행 상황에 따라 표시됩니다. 응답률은 **조합원 수 기준과 출자좌수 기준**으로 확인할 수 있습니다.  
  
#### 총회 관련 문서 확인하기  
  
총회에 등록된 **설명 자료 또는 의결 관련 문서**를 확인할 수 있습니다.  
  
문서를 클릭하면 미리보기 및 다운로드가 가능합니다.  

#### 담당자 정보 확인하기  
  
총회 관련 문의를 위해 **담당자 정보**가 LPworks에 제공됩니다.  
  
확인 가능한 정보: 담당자 이름, 부서, 직책, 이메일, 내선번호
 
---  
  
### 안건 내용 확인하기  
  
총회에 등록된 안건은 **카드 형태로 정리되어 표시됩니다.**  
  
안건은 다음 순서로 확인할 수 있습니다.  
  
1. 기간 단축 동의  
2. 보고 안건  
3. 부의 안건  
  
각 안건에서는 다음 정보를 확인할 수 있습니다.  
  
- 안건 제목  
- 안건 설명  
- 첨부파일  
  
표결이 필요한 안건의 경우 **찬성/반대 의견 결과**도 함께 확인할 수 있습니다.  
  
---  
### 총회 진행하기  
  
기본정보 탭 상단에서는 **총회의 진행 상태에 따라 필요한 작업을 수행할 수 있습니다.**  
사용 가능한 기능은 총회의 상태에 따라 달라집니다.  
  
#### 1) 개최 전 / 예정 상태  
  
총회를 아직 조합원에게 전송하지 않았거나, 개최 예정 상태인 경우입니다.  
  
| 기능    | 설명                                 |
| ----- | ---------------------------------- |
| 수정    | 총회 제목, 일정, 응답 기간, 안건 등의 정보를 수정합니다. |
| 삭제    | 등록된 총회를 삭제합니다.                     |
| 개최 안내 | 조합원에게 총회 개최를 안내합니다.                |
  
**개최 안내**  
  
개최를 안내하면 조합원에게 다음 내용이 전달됩니다.  
  
- 총회 개최 안내  
- 총회 상세 페이지 접속 링크  
- 응답 기간 안내  
  
조합원은 전달받은 링크를 통해 총회에 접속하여  
  
- 안건 확인  
- 찬성/반대 의견 제출  
- 문서 확인  
  
등의 작업을 수행할 수 있습니다.  
  
---  
  
#### 2) 진행중 상태  
  
조합원에게 총회 안내가 전송되고 **응답이 진행 중인 상태**입니다.  
  
| 기능      | 설명                            |     |
| ------- | ----------------------------- | --- |
| 안내사항 전송 | 조합원에게 추가 안내 메시지를 보냅니다.        |     |
| 리마인드 전송 | 아직 응답하지 않은 조합원에게 참여 요청을 보냅니다. |     |
| 마감일 연장  | 서면결의 응답 마감일을 연장합니다.           |     |
| 철회      | 진행 중인 총회를 취소합니다.              |     |
  
**안내사항 전송**  
  
안내사항 전송 기능을 사용하면 조합원에게 다음과 같은 추가 공지를 전달할 수 있습니다.  
  
예시  
  
- 총회 관련 추가 설명  
- 안건 관련 보충 안내  
- 총회 수정사항 안내
  
전송된 안내사항은 조합원이 총회 화면에서 다시 확인할 수 있습니다.  
  
**리마인드 전송**  
  
응답하지 않은 조합원에게 총회 참여를 다시 안내할 수 있습니다.  
이를 통해 총회 응답률을 높일 수 있습니다.  

표결하지 않은 조합원만 목록에 표시됩니다.

**철회**  
  
총회 안내를 철회할 수 있습니다. 철회하면 LPworks에서 총회가 취소되며, 내용을 조회할 수 없습니다.
  
---  
  
#### 3) 마감 상태  
  
서면결의 응답 기간이 종료된 상태입니다.  
  
| 기능     | 설명                         |
| ------ | -------------------------- |
| 결과 입력  | 의사록과 안건별 가결/부결 결과를 등록합니다.  |
| 마감일 연장 | 필요 시 응답 기간을 다시 연장할 수 있습니다. |

**결과 입력**  
  
화면 중앙의 결과 카드에서 결과를 확인하고, 안건 카드에서 안건별 가결/부결 결과를 등록할 수 있습니다.

결과 카드에서는 아래 내용을 확인 가능합니다.   
  
- 보고 안건/부의 안건 수  
- 가결/부결/조건부 가결 건수  
- 응답 조합원 수  
- 응답 출자좌수  
   
결과 카드에서 의사록을 업로드 해야 조합원에게 결과를 전송할 수 있습니다. 
결과의 변동사항이 있으면 결과를 재전송할 수 있습니다.

---  
  
#### 4) 완료 상태  
  
총회 결과가 확정된 상태입니다.  
  
| 기능    | 설명                  |
| ----- | ------------------- |
| 결과 전송 | 총회 결과를 조합원에게 안내합니다. |
  
**결과 전송**  
  
총회 결과를 전송하면 조합원은  
  
- 안건별 가결 여부  
- 총회 결과 요약  
- 결과 문서(의사록)  
  
등을 확인할 수 있습니다.  
  
---  

### 총회 참여 흐름  
  
조합원은 안내 메시지를 받은 후 다음 과정으로 총회에 참여합니다.  
  
1. 안내 메시지 수신  
2. 메시지에 포함된 총회 링크 클릭  
3. 총회 상세 화면 접속  
4. 안건 확인  
5. 찬성 / 반대 의견 제출  
  
조합원 유형에 따라 접속되는 서비스가 다를 수 있습니다.

---

### 전송

#### 총회 안내 전송하기  
  
조합원에게 총회 안내를 보내려면 **전송 모달**을 사용합니다.  
전송 모달은 **Step1과 Step2 두 단계**로 구성되어 있습니다.

---  
##### Step1. 전송 내용 확인하기  

 ![](assets/img/개최%20안내%20모달.png)
  
Step1에서는 조합원에게 전달될 **메시지 내용을 확인합니다.**  
  
확인할 수 있는 내용  
  
- 전송 유형 (개최 안내 / 안내사항 / 리마인드 / 결과 안내)  
- 안내사항 제목   
- 안내사항 내용  
- 포함된 총회 접속 링크  
  
전송되는 메시지에는 **조합원총회에 참여할 수 있는 링크**가 포함됩니다.  
조합원은 해당 링크를 통해 총회에 접속하여 안건을 확인하고 의견을 제출할 수 있습니다.  
안내사항 제목은 이메일의 제목으로도 사용됩니다.
  
내용을 확인한 후 **다음 버튼을 클릭하면 Step2로 이동합니다.**

---  
##### Step2. 전송 대상과 방법 선택하기  
  
![](assets/img/개최%20안내%20모달%202.png)
  
Step2에서는 **메시지를 받을 조합원과 전송 방법을 설정합니다.**  
  
설정할 수 있는 항목  
  
- 전송 대상  
- 전송 채널 (이메일 / 알림톡 / SMS)  
  
시스템은 조합원 유형에 따라 **기본 전송 방법을 자동으로 설정합니다.**  

| 대상 유형    | 기본 전송 방법 | 비고                                         |
| -------- | -------- | ------------------------------------------ |
| 회원 (법인)  | 이메일      | 기본값으로 이메일만 선택됩니다.                          |
| 회원 (개인)  | 알림톡(SMS) | 기본값으로 알림톡(SMS)만 선택됩니다.                     |
| 비회원 (개인) | 알림톡(SMS) | 기본값으로 알림톡(SMS)만 선택됩니다.                     |
| 비회원 (법인) | 전송 불가    | LPworks 및 비회원 링크를 사용할 수 없어 전송 버튼이 비활성화됩니다. |

---

#### 전송 유형 안내  
  
조합원총회에서는 상황에 따라 조합원에게 다양한 안내를 전송할 수 있습니다.  
전송 유형에 따라 **전송 시점과 전달되는 내용이 다릅니다.**   
  
| 전송 유형   | 전송 시점                 | 전달 내용             |
| ------- | --------------------- | ----------------- |
| 개최 안내   | 총회를 처음 안내할 때          | 총회 개최 안내와         |
| 안내사항 전송 | 총회 진행 중 추가 공지가 필요한 경우 | 추가 안내 메시지 및 관련 정보 |
| 리마인드    | 응답하지 않은 조합원이 있는 경우    | 총회 참여 요청 안내       |
| 마감일 연장  | 서면결의 마감일을 연장하는 경우     | 마감일 연장 안내         |
| 결과 전송   | 총회 종료 후               | 총회 결과 및 결과 문서 안내  |
| 철회 전송   | 총회를 철회할 때             | 총회 취소 안내          |

---  
 
#### 전송 채널 안내  
  
총회 안내는 다음 채널을 통해 전송할 수 있습니다.  
  
| 전송 채널 | 설명                          |
| ----- | --------------------------- |
| 이메일   | 조합원 이메일 주소로 안내 메일을 전송합니다.   |
| 알림톡   | 카카오 알림톡으로 총회 안내 메시지를 전송합니다. |
| SMS   | 알림톡 전송 실패 시 문자 메시지를 전송합니다.  |
  
알림톡 전송이 실패하면 **자동으로 SMS가 발송됩니다.**  
  
> 알림톡 및 SMS 전송 시 메시지 발송 비용이 발생할 수 있습니다.  
  
---  
  
#### 전송 대상별 접속 링크  
  
전송 메시지에는 **총회 참여를 위한 링크**가 포함됩니다.  
조합원의 유형에 따라 접속되는 서비스가 다릅니다.  
이메일은 PC, 알림톡(SMS)과 비회원 참여 페이지는 모바일에서 사용 최적화되어 있습니다.
  
| 대상 유형    | 이메일              | 알림톡(SMS)         |     |
| -------- | ---------------- | ---------------- | --- |
| 회원       | LPworks 웹        | LPworks 앱 또는 웹   |     |
| 비회원 (개인) | 비회원 전용 총회 참여 페이지 | 비회원 전용 총회 참여 페이지 |     |
  
조합원은 전달받은 링크를 통해 총회에 접속하여  
  
- 총회 안내 확인  
- 안건 내용 확인  
- 찬성/반대 의견 제출  
  
등의 작업을 수행할 수 있습니다.

---  
  
## 자주 묻는 질문  

> 총회 안내를 전송하면 무엇이 조합원에게 전달되나요?  
{: .prompt-tip }  
  
- 총회 안내가 전송되면 조합원에게 **총회 안내 메시지와 총회 접속 링크**가 전달됩니다.  
조합원은 해당 링크를 통해 총회 상세 화면에 접속하여 안건을 확인하고 의견을 제출할 수 있습니다.
  
> 응답률은 언제부터 표시되나요?  
{: .prompt-tip }  
  
- 총회가 **진행중 상태가 되면 응답률이 표시됩니다.**  
  
> 총회 문서는 전송 이후에도 수정할 수 있나요?  
{: .prompt-tip }  
  
- 총회를 조합원에게 **최초 전송한 이후에는 문서를 수정할 수 없습니다.**

> 응답 기간이 끝났는데 다시 참여를 받을 수 있나요?  
{: .prompt-tip }  
  
- 총회 상태가 **마감 상태인 경우에도 마감일 연장 기능을 통해 응답 기간을 다시 열 수 있습니다.**

> 비회원 법인 조합원에게는 왜 전송할 수 없나요?  
{: .prompt-tip }  
  
- 현재 비회원 법인은 LPworks 계정이 없고 비회원 참여 링크도 제공되지 않아 총회 참여 링크를 전달할 수 없습니다. 비회원 법인도 참여할 수 있도록 보완할 예정입니다.

> 알림톡을 선택하면 SMS도 함께 발송되나요?  
{: .prompt-tip }  
  
- 알림톡 전송이 실패한 경우에만 **자동으로 SMS가 발송됩니다.**

> 메시지 전송 비용이 발생하나요?  
{: .prompt-tip }  
  
- **알림톡과 SMS 발송 시 메시지 발송 비용이 발생할 수 있습니다.**
