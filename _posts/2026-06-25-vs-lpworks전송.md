투자심사에 등록된 세부사항을 LPworks로 전송할 수 있습니다. 전송은 투자심의위원회(투자심사-조합) 단위로 이뤄지며, 투표상태에 따라 계획보고 혹은 결과보고로 전송됩니다.

## 설명

### 화면 경로
`VCworks` → `투자/회수` → `투자 심의` → `투자심사 목록` ( → `투자심사 상세` )

### LPworks로 투자심사 전송하기
#### 목록 화면에서 전송하기
투자심사 목록 화면에서 투자심의위원회를 LPworks로 전송하는 방법은 다음과 같습니다.
![이미지](/assets/img/vs0001_전송.jpg)
1. 투자심사 목록 화면 우측 상단의 [LPworks 전송] 버튼을 클릭해주세요.
2. `LPworks 투자심사 보고 전송` 모달에서 전송하고자 하는 투자심의위원회(조합 단위)을 선택해주세요.
	- 보고단계별 투자심사 상세 탭 필수 입력 값이 누락된 행은 선택할 수 없습니다.
	- `전송중`인 행은 선택할 수 없습니다.
3. [전송] 버튼을 클릭해주세요.
	- 선택한 투자심사 건의 [회사정보, 담당심사역, 투자정보, 본투자심의, 첨부파일] 탭에 포함된 세부항목의 정보 및 첨부파일이 LPworks로 전송됩니다.
	- 선택된 건에 포함된 조합의 `조합 정보`, `출자/배분 및 지분양수도` 내역도 함께 전송됩니다.

##### 보고단계 전송 프로세스
재원별 투자심의위원회의 `투표상태`에 따라, 보고단계가 구분됩니다.
- `계획보고`: 투표상태가 `투표전` 혹은 `투표중`인 경우입니다. 계획보고 열 값만 존재합니다.
- `결과보고`: 투표상태가 `투표종료`인 경우입니다. 계획보고와 결과보고 열 값이 모두 존재합니다.

현재 보고단계와 관계없이 여러 번 전송이 가능합니다.
- `계획보고` 전송 이후 투표상태가 `투표종료`로 바뀐 경우, `결과보고` 전송이 가능합니다.
- `계획보고` 전송을 생략하고 투표상태가 `투표종료`가 된 경우, `결과보고` 전송만 가능합니다.
- 동일한 투자심의위원회에 대해서 LPworks에서는 하나의 행으로 조회됩니다.
	- 최종 전송된 내역만 LPworks에서 조회됩니다.
##### 투자심사 상세 탭 필수 입력 값
투자심사 전송을 위해 필요한 투자심사 상세의 각 탭 내 필수 입력 값은 다음과 같습니다.
필수 입력 값이 모두 입력된 행만 선택이 가능합니다.

- [회사정보]탭
	- `국가`, `사업자번호`, `표준산업분류`, `자본금`, `설립일자`
- [담당심사역]탭
	- `담당자`
- [투자정보]탭
	- 투자조건
		- 투자 유형
			- 투자 유형별 투자 조건의 필수값
		- 기타투자 정보
			- `조합별 특수관계인 거래 여부`
			- `ESG 평가 여부`
	- 동반 투자 조건(존재 시)
		- 동반 투자 개요: `투자 유형`, `세부 투자 유형`
		- 동반 투자 조건: `투자 통화`, `투자 금액`, `주식 수`(투자 유형이 주식일 때만), `주당 단가`(투자 유형이 주식일 때만), `신주/구주`(투자 유형이 주식일 때만)
	- 계약 정보
		- `계약일`, `납입일`
	- 투자 라운드 정보
		- `투자단계`, `기업가치pre`, `기업가치post`
	- 기타 정보
		- `EXIT 계획`, `투자금 사용용도`, `의무불이행에 따른 제재 관련 이자율`, `투자금 사용용도 위반시 제재 방안`
-  [본투자심의]탭
	- `개최일시`, `장소`
	- 보고단계가 `결과보고`인 경우, 위에 더해 투표결과 값이 `가결` 혹은 `부결`
- [첨부파일]탭
	- `투자심사보고서`, `준법사항체크리스트` 첨부
	- 보고단계가 `결과보고`인 경우, 위에 더해 `투심위의사록` 첨부

##### 'LPworks 투자심사 보고 전송' 모달 내 상태 값
`LPworks 투자심사 보고 전송` 모달 내 계획보고 및 결과보고 열의 상태 값에 대한 설명입니다.

| 상태  | 설명                                                               |
| --- | ---------------------------------------------------------------- |
| 가능  | 해당 투자심의위원회가 필수 입력값이 모두 입력되어 있으며, LPworks 전송 이력이 없는 경우입니다.        |
| 불가능 | 해당 투자심의위원회가 필수 입력값이 모두 입력되어 있지 않으며, LPworks 전송 이력이 없는 경우입니다.     |
| 완료  | 해당 투자심의위원회에 대한 LPworks 전송 이력이 있는 경우입니다.                          |
| 만료  | 해당 투자심의위원회의 보고단계가 `결과보고`이며, `계획보고` 단계에서 LPworks 전송 이력이 없는 경우입니다. |
- `가능`인 행은 모달 진입 시, 기본 선택되어 있습니다.
- `불가능`인 행은 선택이 불가합니다.
	- `투자기업명` 셀을 클릭해 해당 투자심사 상세페이지로 이동하여 수정하실 수 있습니다.
- `완료`, `만료`는 각 보고단계에서의 LPworks 전송 이력 존재 여부만 안내합니다. 행 선택 가능 여부에 영향을 주지 않습니다.
#### 상세 화면에서 전송하기
특정 투자심사 상세 화면에서 LPworks 전송하는 경우, 투자심사 목록에서 전송하는 경우와 다른 점은 2가지 입니다.
- `LPworks 투자심사 보고 전송` 모달에 해당 투자심사에 속한 투자심의위원회만 선택지로 등장합니다.
- `투자기업명` 셀을 클릭해도 투자심사 상세 화면으로 이동하지 않습니다.
이 외의 방법은 목록 화면에서 전송하는 방법과 모두 동일합니다.

#### LPworks 데이터 전송 화면에서 전송하기
LPworks 데이터 전송 화면에서도 LPworks로 투자심사 건을 전송할 수 있습니다.
자세한 설명은 [LPworks 데이터 전송]()를 참고해주세요.


## 자주 묻는 질문

> 투자심사는 어떤 단위로 LPworks에 전송되나요?
{: .prompt-tip }
- 투자심사는 `투자심의위원회(투자심사-조합)` 단위로 전송됩니다.
- 하나의 투자심사에 여러 조합이 포함되어 있으면, 조합별 투자심의위원회를 각각 선택해 전송할 수 있습니다.
- LPworks에서는 투자심의위원회당 하나의 행으로 조회됩니다.

> 투자심사 목록과 `LPworks 투자심사 보고 전송` 모달에 등장하는 선택지가 달라요.
{: .prompt-tip }
- `LPworks 투자심사 보고 전송`에는 현재 심의 단계가 '본투자심의' 이상인 선택지만 등장합니다.
- 이미 `투자완료` 이거나, '본투자심의'가 개최되지 않은 투자심사 건은 등장하지 않습니다.

> `계획보고`를 먼저 보내지 않았는데 `결과보고`를 보낼 수 있나요?
{: .prompt-tip }
- 투표상태가 `투표종료`가 된 경우에는 `결과보고` 전송 대상이 됩니다.
- 이 경우 계획보고는 더 이상 전송할 수 없어 `만료` 상태로 보일 수 있습니다.
- 전송 가능 여부는 결과보고 기준 필수 입력값이 모두 입력되어 있는지 확인해 주세요.

> `결과보고` 전송 시 `계획보고`보다 추가로 필요한 값이 있나요?
{: .prompt-tip }
- [본투자심의] 탭에서 투표결과가 `가결` 또는 `부결`이어야 합니다.
- [첨부파일] 탭에 `투심위의사록`이 추가로 첨부되어 있어야 합니다.
- 그 외 계획보고 단계에서 필요한 필수 입력값도 모두 입력되어 있어야 합니다.

> 같은 투자심의위원회를 여러 번 전송하면 LPworks에는 어떻게 보이나요?
{: .prompt-tip }
- 동일한 투자심의위원회는 LPworks에서 하나의 행으로 조회됩니다.
- 여러 번 전송할 수 있지만, LPworks에서는 최종 전송된 내역만 조회됩니다.

> 목록 화면에서 전송하는 것과 상세 화면에서 전송하는 것은 무엇이 다른가요?
{: .prompt-tip }
- 목록 화면에서만 여러 투자심사에 속한 투자심의위원회를 선택할 수 있습니다.
- 상세 화면에서는 현재 투자심사에 속한 투자심의위원회만 선택지로 표시됩니다.

> 전송 중인 행은 왜 다시 선택할 수 없나요?
{: .prompt-tip }
- 이미 LPworks로 전송 요청이 진행 중인 행이기 때문입니다. 
- 전송이 완료된 뒤 다시 모달을 열어 상태를 확인해 주세요.
- 전송 결과는 LPworks 전송 이력 화면에서 확인할 수 있습니다.

> 검토중단된 투자심사는 어떻게 전송하나요?
{: .prompt-tip }
- 별도의 전송 없이, 1건이라도 투자심의위원회 전송 시에 모든 검토중단된 투자심사 건의 정보도 함께 전송됩니다.
-
