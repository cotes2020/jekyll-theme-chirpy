조합원총회 목록 화면은 **운용사가 조합원총회를 등록하고, 진행 상태와 응답 현황을 관리하며,**  
**각 안건의 의결 결과까지 확인**할 수 있는 관리 화면입니다.  
  
등록된 모든 총회를 한눈에 조회할 수 있으며, 검색·필터·정렬 기능을 활용해 원하는 총회를 빠르게 찾을 수 있습니다.

---

## 설명
### 총회 목록 구성  
  
화면에는 등록된 조합원총회가 목록 형태로 표시됩니다.  
각 행은 하나의 조합원총회를 의미합니다.  
  
**표시 항목**  
 
- **진행상태**  
- 개최전 / 예정 / 철회 / 진행중 / 마감 / 완료 / 등록  
- **제목**  
- 조합원총회 제목  
- **조합**  
- 해당 총회를 진행하는 조합  
- **총회 유형**  
- 결성 총회 / 정기 총회 / 임시 총회 / 해산 총회 / 청산 총회 / 간담회 / 보고회 / 기타  
- **개최일**  
- 총회 개최일  
- **응답률**  
- 전체 출자좌수 대비 응답 비율  
- 소수점 둘째 자리까지 표시   

기본 정렬은 **등록일시 기준 내림차순(최신순)** 입니다.

---  
### 안건 확인 방법  
  
총회 행을 클릭하면  
해당 총회에 포함된 안건 목록이 펼쳐집니다.  
  
안건에는 다음 정보가 표시됩니다.  
  
- 안건 구분 (보고 안건 / 부의 안건)  
- 안건명  
- 안건 유형  
- 결의 방법  
- 결의 결과 (가결 / 부결 / 조건부 가결) 
- 찬성 좌수
- 찬성률
  
안건을 클릭하면 조합원총회 상세 화면으로 이동합니다.  
  
---  
  
### 신규 등록 기능  
  
우측 상단의 **[신규 등록] 버튼**을 통해 조합원총회를 생성할 수 있습니다.  
  
신규 등록에는 두 가지 방식이 있습니다.  

- **총회 개최용 등록**  
	- 시스템을 통해 조합원에게 안내문을 보내고  
	- 서면결의를 받고  
	- 결과를 확정하는 경우에 사용합니다.  
	- 앞으로 열릴 총회를 준비할 때 선택합니다.  
	- 저장 시 진행상태는 **‘개최전’** 으로 설정됩니다. 
	- 이후 전송 → 응답 수집 → 결과 확정 절차로 진행됩니다.  
  
- **총회 기록용 등록**  
	- 이미 오프라인이나 다른 방식으로 끝난 총회를  
	- 내부 관리 목적상 남겨두는 경우입니다.  
	- 조합원에게 별도 전송은 하지 않습니다. 
	- 저장 시 진행상태는 **‘등록’** 상태로 저장됩니다. 
	- LPworks를 통한 응답 수집은 진행되지 않습니다.  

---  
  
### 진행상태 안내  
  
- 개최전: 총회 생성 후 아직 진행되지 않은 상태  
- 예정: 표결이 시작되지 않은 상태  
- 진행중: 조합원 응답을 수집 중인 상태  
- 마감: 응답 마감 처리 완료  
- 완료: 결의 결과 확정 완료  
- 철회: 총회 취소  
- 등록: 기록 목적의 총회 데이터  
  
---  
  
### 응답률  
  
응답률은 **조합원수 기준**으로 계산됩니다.  
전송 여부와 무관하게 전체 조합원 모수로 산정됩니다.  
  
#### 응답률이 표시되지 않는 상태  
  
- 개최전  
- 예정  
  
---  
  
## 자주 묻는 질문  
  
> 총회를 클릭했는데 바로 상세 화면으로 이동하지 않습니다.  
{: .prompt-tip }  
- 총회 행을 클릭하면 먼저 안건 목록이 펼쳐집니다.  
- 안건을 클릭해야 상세 화면으로 이동합니다.  
  
> 응답률은 전송한 조합원 기준인가요?  
{: .prompt-tip }  
- 맞습니다. 응답률은 전체 조합원 기준으로 계산됩니다.  
  
> ‘등록’ 상태는 무엇인가요?  
{: .prompt-tip }  
- 이미 종료된 총회를 기록 목적으로 등록한 데이터입니다.
