조합원총회 상세의 **안건상세 탭**에서는 총회에 등록된 안건을 확인하고, 안건별 표결 현황과 조합원 응답 상태를 조회할 수 있습니다.  
또한 필요 시 운용사가 조합원의 의견을 **직접 등록** 할 수 있습니다.

---

## 설명  
  
![](assets/img/agendadetail.png)
### 1. 안건 선택하기  
좌측의 **안건 목록**에서 확인할 안건을 선택합니다.  
  
- 안건은 **보고 안건 → 부의 안건 순서**로 표시됩니다.  
- 안건을 클릭하면 우측에 **선택한 안건의 상세 정보와 표결 현황**이 표시됩니다.  
  
---  
  
### 2. 안건 내용 확인하기  
선택한 안건의 상세 내용을 확인할 수 있습니다.  
  
확인 가능한 정보는 다음과 같습니다.  
  
**보고 안건**  
  
- 안건명  
- 상세 설명  
- 첨부파일  
  
**부의 안건**  
  
- 안건명  
- 안건 유형  
- 결의 방법  
- 관리보수 삭감비율  
- 상세 설명  
- 첨부파일
  
---  
  
### 3. 안건 표결 현황 확인하기  
안건에 대한 **조합원의 표결 결과**를 확인할 수 있습니다.  
  
- **투표 내역 차트**  
- 조합원의 **출자 비율 기준**으로 결과 표시  
	- 출자 비율은 **잔여 출자 좌수**를 기준으로 계산
- 찬성 / 반대 / 기권 / 미응답 비율 확인  
  
- **조합원 응답 현황**  
- 조합원별 표결 결과 및 의견 확인  

---
### 4. 조합원 의견 직접 등록하기  
  
조합원이 LPworks 또는 비회원 링크를 통해 의견을 제출하지 않은 경우  
운용사가 조합원의 의견을 **직접 등록**할 수 있습니다.  
  
**등록 방법**  
  
1. 화면 상단 **[의견 직접등록] 버튼** 클릭  
2. 의견 직접등록 모달에서 **조합원 선택**  
3. 안건별 **찬성 / 반대 / 기권 및 의견 입력**  
4. **서면결의서** 등 첨부파일 첨부
5. **[저장] 버튼 클릭**  

#### 의견 직접등록 화면 구성  
![](assets/img/ansmodal.png)  
##### 상단 정보  
  
| 항목      | 설명                               |
| ------- | -------------------------------- |
| 조합원     | 의견을 등록할 조합원 선택                   |
| 의견제출 여부 | 해당 조합원이 LPworks를 통해 의견을 제출했는지 여부 |
| 본인인증 여부 | 조합원이 본인 인증을 완료했는지 여부             |
| 수신문서    | 조합원이 제출한 문서                      |
  
조합원이 이미 시스템을 통해 응답했는지 확인할 수 있습니다.  
  
##### 의견 입력 영역  
  
| 항목   | 설명              |
| ---- | --------------- |
| 안건구분 | 보고 안건 / 부의 안건   |
| 안건명  | 안건의 제목          |
| 찬/반  | 찬성 / 반대 / 기권 선택 |
| 의견   | 조합원의 의견 입력      |
 
##### 직접 등록시 참고사항  
  
- 조합원별로 **안건마다 찬성 / 반대 / 기권 및 의견을 입력**할 수 있습니다.  
- 조합원이 **LPworks 또는 비회원 링크로 이미 의견을 제출한 경우** 해당 내용은 **VCworks에서 수정할 수 없습니다.**  
- 직접 등록한 의견은 **필요 시 다시 수정할 수 있습니다.**  
- 의견 직접등록은 **조합원의 회신을 대신 기록하기 위한 기능**이며, 수신문서 탭의 회신 여부와는 별개로 관리됩니다.
	- 다만, 첨부파일은 수신 문서 탭과 공유됩니다.
  
---  
  
## 자주 묻는 질문  
  
> 조합원이 이미 LPworks에서 의견을 제출했는데 VCworks에서 수정할 수 있나요?  
{: .prompt-tip }  
  
- LPworks 또는 비회원 링크를 통해 제출된 의견은 **VCworks에서 수정할 수 없습니다.**  
시스템에서 직접 등록한 의견보다 **LPworks 회신 내용이 우선 적용**됩니다.  
  
> 의견 직접등록을 하면 수신문서 탭의 회신으로 처리되나요?  
{: .prompt-tip }  
  
- 아닙니다. **의견 직접등록은 회신 처리와 별개 기능**입니다.  
수신문서 탭의 회신 여부와는 무관하게 의견을 기록하기 위한 기능입니다.  
  
> 총회가 종료된 후에도 의견을 직접 등록할 수 있나요?  
{: .prompt-tip }  
  
- 의견 직접등록은 **총회 상태가 진행중 또는 마감일 때만 가능**합니다.  
종료된 총회에서는 등록할 수 없습니다.
