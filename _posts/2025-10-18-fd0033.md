조합원총회의 대상이 되는 **조합원 명부와 회신 문서를 확인할 수 있는 화면**입니다.  
조합원별 개최 안내 여부, 회신 여부, 결과 안내 여부를 확인할 수 있으며 조합원이 제출한 문서를 조회하거나 다운로드할 수 있습니다.

---
## 설명  

![](assets/img/Pasted%20image%2020260310185304.png)
  
수신문서 탭에서는 총회 기준일을 기준으로 한 **조합원 명부와 회신 문서 현황**을 확인할 수 있습니다.  
또한 조합원이 제출한 문서를 개별 다운로드하거나 여러 문서를 **일괄 다운로드**할 수 있습니다.  
  
---  
  
### 1. 조합원 명부 기준  
  
수신문서 탭의 조합원 목록은 **총회 등록 시 설정된 기준일을 기준으로 작성된 조합원 명부**를 사용합니다.  
  
- 총회 대상 조합원 목록 표시  
- 하단 합계 영역에서 전체 조합원 수 및 출자좌수 합계 확인 가능  
- 출자 좌수는 **잔여 출자 좌수** 기준
  
---  
  
### 2. 개최 안내 / 회신 / 결과 안내 여부  
  
다음 3개의 항목을 통해 조합원별 진행 상태를 확인할 수 있습니다.  
  
| 항목       | 표시 기준                                             |
| -------- | ------------------------------------------------- |
| 개최 안내 여부 | 조합원에게 이메일, 알림톡(SMS) 등 **1회 이상 개최 안내를 전송한 경우 `여`** |
| 회신 정보 여부 | 조합원이 **LPworks 또는 비회원 응답 링크를 통해 회신한 경우 `여`**      |
| 결과 안내 여부 | 조합원에게 이메일, 알림톡(SMS) 등 **1회 이상 결과 안내를 전송한 경우 `여`** |
  
- 제공 값: `여`, `부`  
- VCworks에서 **직접 입력한 의견은 회신으로 계산되지 않습니다.**  
  
---  
  
### 3. 수신문서 컬럼 생성 방식  
  
수신문서 영역은 **총회 등록 시 선택한 회신문서 유형에 따라 동적으로 생성**됩니다.  
  
예를 들어 총회 등록 시 다음 문서를 회신필요로 선택한 경우  
  
- 서면결의서  
- 인감증명서  
- 위임장  
- 기타자료 동의서  
  
수신문서 탭의 표에도 동일한 이름의 **열이 생성됩니다.**  
  
---  
  
### 4. 수신문서 다운로드  
  
조합원이 제출한 문서는 개별적으로 다운로드할 수 있습니다.  
  
**다운로드 방법**  
  
1. 수신문서 탭 이동  
2. 문서가 첨부된 셀 클릭  
3. 파일 미리보기 확인  
4. 다운로드 또는 인쇄  
  
---  
  
### 5. 수신문서 일괄 다운로드  
  
여러 조합원의 문서를 한 번에 다운로드할 수 있습니다.  
  
**사용 방법**  
  
1. 조합원 선택  
2. **[일괄 다운로드] 버튼 클릭**  
3. 다운로드할 문서 유형 선택  
4. **[다운로드] 클릭**  
  
일괄 다운로드 모달에서는 **총회 등록 시 수신문서로 선택한 문서 유형만 선택할 수 있습니다.**  
  
---  
  
### 6. 일괄 다운로드 파일 구조  

일괄 다운로드 시 문서는 ZIP 파일로 제공됩니다.  
  
ZIP 파일 내부에는 **문서 유형별 폴더가 생성되며, 각 폴더 안에 조합원별 문서가 포함됩니다.**

예시

총회유형_날짜.zip  
├ 서면결의서  
│ ├ 벤처캐피탈A_서면결의서.pdf  
│ ├ 벤처캐피탈B_서면결의서.pdf  
│  
├ 위임장  
│ ├ 벤처캐피탈A_위임장.pdf  
│ ├ 벤처캐피탈C_위임장.pdf  
│  
├ 기타자료 동의서  
│ ├ 벤처캐피탈B_기타자료동의서.pdf

  
파일명 구성 기준  
  
- ZIP 파일명: **총회유형_날짜**  
- 폴더명: **문서 유형명**  
- 파일명: **조합원명_원본파일명**

## 자주 묻는 질문

> 수신문서 컬럼은 어떤 기준으로 생성되나요?
{: .prompt-tip }

- 총회를 등록할 때 **회신 필요 문서로 선택한 문서의 수만큼 컬럼이 생성되며, 문서 이름이 그대로 컬럼 제목으로 표시됩니다.**

> 회신 정보 여부는 어떤 경우에 `여`로 표시되나요?
{: .prompt-tip }

- 조합원이 **LPworks 또는 비회원 응답 링크를 통해 총회에 응답한 경우 `여`로 표시됩니다.** VCworks에서 관리자가 직접 등록한 의견은 회신으로 계산되지 않습니다.

> 일괄 다운로드에서 선택할 수 있는 문서는 무엇인가요?
{: .prompt-tip }

- 총회 등록 시 **수신문서로 선택한 문서 유형만 다운로드 대상 문서로 표시됩니다.**

> 회신 정보 여부가 `여`인데 수신문서가 없는 경우가 있습니다.
{: .prompt-tip }

- 회신 정보 여부는 **조합원이 총회에 대해 응답(찬성/반대 등)을 했는지 여부**를 기준으로 표시됩니다.  
- 따라서 조합원이 LPworks 또는 비회원 링크를 통해 총회에 응답했지만 문서를 첨부하지 않은 경우 **회신 정보 여부는 `여`로 표시되지만 수신문서는 없을 수 있습니다.**
