## 설명
- 진입 경로: `조합 > 조합원 증명서 발급`
- 본 기능은 LPworks로부터 수신한 조합원 증명서 발급 요청 건에 대한 회신만을 다룹니다. 조합원 증명서 출력 자체에 대한 내용을 파악하려면 [조합원 명부/출자 증서/출자 확인서 출력](/posts/pr0011/)을 참고해주세요.

#### 증명서 발급 요청 회신하기
- 조합원이 요청한 증명서 발급 건에 대해 회신하는 방법은 다음과 같습니다.

1. 먼저, VCworks 알림을 통해 조합원의 요청을 수신합니다.
2. 해당 알림을 클릭하여 `조합원 증명서 발급` 메뉴로 진입합니다.
3. `회신필요` 상태의 행을 클릭합니다.
4. '증명서 발급 회신' 드로어에서 요청 정보를 확인합니다.
5. 증명서 첨부 방식을 선택합니다.
	- 문서 자동생성: 요청받은 유형의 증명서를 시스템에서 생성합니다.
	- 직접 파일첨부: 직접 회신파일을 첨부합니다.
6. 문서 자동생성을 선택한 경우, [생성] 버튼을 클릭합니다.
	- 직접 파일첨부를 선택한 경우, [파일 선택] 버튼을 클릭합니다.
7. 문서 자동생성을 선택한 경우, [미리보기] 버튼을 클릭하여 생성된 증명서를 검토합니다.
	- 직접 파일첨부를 선택한 경우, 파일 첨부 후 필요 시 첨부한 파일을 클릭해 검토할 수 있습니다.
8. [회신] 버튼을 클릭하여 생성된(첨부한) 증명서를 회신합니다.


## 자주 묻는 질문

> 파일 첨부가 안돼요.
> {: .prompt-tip }

- 이미 첨부하거나 생성된 파일이 존재하는 경우에 파일 첨부는 제한됩니다.
- 기존에 첨부하거나 생성된 파일 옆 휴지통 아이콘을 클릭해 삭제 후, 다시 첨부해주세요.
    

> 인감 정보가 없으면 문서 자동생성이 불가능한가요?
> {: .prompt-tip }

- 네, 조합원 증명서 문서 자동생성을 위해서는 조합인감이 필요합니다.
- `조합 > 조합 정보 > 조합 클릭 > 필요 서류 탭`에서 조합인감 이미지를 등록해주세요.
    

> 문서 워터마크가 없으면 문서 자동생성이 불가능한가요?
> {: .prompt-tip }

- 아니요, 조합원 증명서 문서 자동생성을 위해서 문서 워터마크 정보는 필수적이지 않습니다.
- [미리보기]를 통해 문서 워터마크가 없이 생성된 조합원 증명서를 확인하실 수 있습니다.
- 문서 워터마크 출력을 원하시면, `설정 > 회사 정보 관리 > 회사 정보 > 기본 정보 섹션`에서 문서 워터마크 이미지를 등록해주세요.
    

> 회신기한이 지나면, 회신이 불가한가요?
> {: .prompt-tip }

- 아니요, 회신기한이 지나도 회신이 가능합니다.
- 다만, 회신기한이 지난 경우에는 회신 전까지 LPworks에서 해당 요청 행의 상태가 `기한경과`로 표기됩니다.
	- 회신 완료 시, LPworks에 `발급완료`로 표기됩니다.
