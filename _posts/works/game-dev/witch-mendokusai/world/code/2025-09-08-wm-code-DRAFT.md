## 언어 & 코드

---

주문을 이루는 문자와 문법  

각 국가마다 상징되는 언어가 있음  
일상 언어와는 차이가 있음  

각 언어는 '패러다임'을 가지고 있고, 특정한 목적을 가짐  

### 고대 언어

- 다루기 어렵지만, 극도로 효율적임
  - 마나의 흐름에 직접 개입
  - 작은 실수에도 폭주할 수 있음

### 공용어

- 다루기 쉽지만, 비효율적임
  - 자연어에 가까움
  - 즉흥적이거나 가벼운 일상 마법 구현에 주로 사용됨
  - 세계 공용 언어
    - 널리 퍼진 바이블이 있음

### 제국어

- 적당히 다루기 어렵고, 적당히 효율적임
  - 제국의 언어
  - 속성과 행동을 가진 객체라는 개념을 다룸
  - 대규모 전쟁에 유리함
    - 복잡하고 거대한 마법을 다루는 것에 절대적인 강점이 있음

### 목적어

- 특정 목적을 다루기 위한 언어

### 비밀어

- 특정 집단이 서로만 알아볼 수 있도록 설계한 언어

### 룬

- 영구적인 마법? 인챈트?

## 주문서

---

주문을 적는 얇은 면  

지역마다, 문화마다 주문서의 모양은 천차만별임  

종일 낮인 우에는 흰 면에 검은 문자를 적는 것이 일반적임  
종일 밤인 아에는 검은 면에 흰 문자를 적는 것이 일반적임  
어떤 곳은 파란 배경에 밝은 파랑 문자를 적기도 하고, 어떤 곳은 붉은 배경에 분홍 문자를 적기도 함.  

국가마다, 가문마다, 개인마다 다른 형태의 주문서를 가짐
