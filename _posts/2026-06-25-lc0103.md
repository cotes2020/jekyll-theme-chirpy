LPworks로 전송한 데이터의 전송 이력을 확인할 수 있는 기능에 대한 설명입니다.

## 설명
- 'LPworks 전송 이력'에서는 각 전송 화면에서 LPworks로 전송한 이력을 전송 데이터 항목별로 확인할 수 있습니다.
- 동일한 전송 단위의 데이터는 여러 화면에서 동시에 전송되지 않도록 관리되며, 하나의 전송 이력으로 확인할 수 있습니다.

### 화면 경로
`VCworks` → `LP보고` → `LPworks` → `전송 이력`

#### LPworks 전송 이력 확인하기
- LPworks로 전송한 데이터 이력을 확인하는 방법은 다음과 같습니다.
![이미지](/assets/img/lc0103.jpg)

1. 'LPworks 전송 이력' 화면에서 전송 이력 목록을 확인해 주세요.
2. 필요한 경우 필터를 사용해 `대상 조합`, `전송 데이터`, `등록 상세`, `전송 결과`, `전송자` 등을 기준으로 이력을 조회해 주세요.
3. `등록 상세` 셀을 클릭하면 해당 데이터를 등록한 상세 화면으로 이동할 수 있습니다.

##### 전송 이력 목록
- LPworks 전송 이력 목록에서는 다음 정보를 확인할 수 있습니다.

| 항목     | 설명                                                   |
| ------ | ---------------------------------------------------- |
| 전송 일시  | 각 전송 화면에서 전송 과정 상 마지막 사용자 행동을 진행한 일시입니다.             |
| 대상 조합  | 전송 단위에 해당하는 조합명입니다.                                  |
| 전송 데이터 | LPworks로 전송한 데이터명입니다.                                |
| 구분     | 조합과 전송 데이터명만으로 특정하기 어려운 전송 건을 구분하기 위한 값입니다.          |
| 등록 상세  | 전송 데이터를 등록한 상세 화면입니다. 셀을 클릭하면 해당 상세 화면으로 이동할 수 있습니다. |
| 전송 결과  | LPworks 전송 결과입니다.                                    |
| 전송자    | 각 전송 화면에서 전송 과정 상 마지막 사용자 행동을 진행한 사용자입니다.            |

##### 전송 결과
- LPworks 전송 이력의 `전송 결과`는 다음 중 하나의 값을 가지게 됩니다.

| 상태 | 설명 |
| --- | --- |
| 성공 | 전송 건이 LPworks에 정상적으로 전송된 상태를 의미합니다. |
| 실패 | 전송 건이 LPworks에 정상적으로 전송되지 않은 상태를 의미합니다. |
| 전송중 | 전송 건이 LPworks에 전송 중인 상태를 의미합니다. |

#### 각 전송 화면에서 전송 이력으로 진입하기

##### 모듈별 전송 화면에서 이동한 경우
- 다음 화면에서 `최근 전송일시` 셀을 클릭하면 'LPworks 전송 이력' 화면으로 진입할 수 있습니다.

1. '출자/배분 계획 목록'
	- `조합` → `출자/배분` → `출자/배분 계획`
2. '투자심사 목록'의 'LPworks 투자심사 보고 전송' 모달
	- `투자/회수` → `투자 심의`
3. '투자심사 상세'의 'LPworks 투자심사 보고 전송' 모달
	- `투자/회수` → `투자 심의` → `투자 심의 상세`

- 위 화면에서 진입한 경우, 클릭한 전송 건과 일치하는 전송 이력 행이 강조 표시됩니다.
	- 다른 행을 클릭하면 강조 표시가 해제됩니다.

##### LPworks 데이터 전송 화면에서 이동한 경우
- 'LPworks 데이터 전송' 화면의 `대상 조합 선택` 섹션에서 각 행의 `최근 전송일시` 셀을 클릭하면 'LPworks 전송 이력' 화면으로 이동할 수 있습니다.

- 'LPworks 데이터 전송' 화면
	- `LP보고` → `LPworks` → `데이터 전송`

- 이 경우, 클릭한 행의 조합과 동일한 `대상 조합` 필터가 적용된 상태로 전송 이력 화면에 진입합니다.

## 자주 묻는 질문

> 어떤 화면에서 보낸 전송 이력을 확인할 수 있나요?
> {: .prompt-tip }
- 'LPworks 데이터 전송' 화면에서 보낸 이력을 확인할 수 있습니다.
- 출자/배분 계획, 투자심사 보고, 영업보고서, 수시보고처럼 각 업무 화면에서 LPworks로 전송한 이력도 함께 확인할 수 있습니다.


> 전송 결과가 '전송중'으로 표시돼요.
> {: .prompt-tip }
- `전송중`은 해당 전송 건이 LPworks에 전송 중인 상태를 의미합니다.
- 전송 처리가 완료되면 결과가 `성공` 또는 `실패`로 변경됩니다.


> 등록 상세를 클릭하면 어디로 이동하나요?
> {: .prompt-tip }
- 해당 전송 데이터를 등록한 상세 화면으로 이동합니다.
- 예를 들어 `출자/배분 계획` 전송 건은 `출자/배분 계획 상세`로, `투자심사 보고` 전송 건은 `투자심사 상세`로 이동할 수 있습니다.
- 등록 상세가 없는 전송 데이터는 상세 이동 대상이 표시되지 않을 수 있습니다.


> 구분 값은 왜 표시되나요?
> {: .prompt-tip }
- 같은 조합과 같은 전송 데이터라도 보고 기준일, 투자기업명, 분기, 기준일 등에 따라 서로 다른 전송 건일 수 있습니다.
- `구분`은 사용자가 전송 건을 구분할 수 있도록 표시되는 보조 정보입니다.


> LPworks 데이터 전송 화면에서 들어왔는데 특정 조합만 보여요.
> {: .prompt-tip }
- 'LPworks 데이터 전송' 화면에서 조합별 `최근 전송일시`를 클릭해 진입한 경우, 해당 조합이 `대상 조합` 필터로 적용됩니다.
- 전체 이력을 보고 싶다면 `대상 조합` 필터를 해제해 주세요.


> 최근 전송일시를 클릭해서 들어왔는데 한 행이 강조되어 있어요.
> {: .prompt-tip }
- 출자/배분 계획 목록, 투자심사 목록, 투자심사 상세 등 모듈별 전송 화면에서 `최근 전송일시`를 클릭해 진입한 경우입니다.
- 클릭한 전송 건과 일치하는 전송 이력 행이 강조 표시됩니다.
