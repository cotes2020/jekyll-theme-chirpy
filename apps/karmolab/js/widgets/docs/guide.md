# 사용자 가이드

## 랜딩 페이지

**홈**에서 모든 도구를 카드로 한눈에 볼 수 있어요.

- **기능**: 암호화, 메모장, AI 챗봇, 이미지 생성 등
- **놀이**: 가챠, 운세, 쓰다듬기, 파티클 등
- **도구**: 비밀번호, 모스 부호, 카운트다운 등

상단 **검색 바**에 도구 이름이나 설명을 입력하면 필터링돼요.

## 마스코트 (티메토)

화면 오른쪽 아래에 **연구소장 티메토**가 있어요.

- **클릭**: 랜덤 대사 + 호감도 1 상승
- **도구 사용**: 도구마다 다른 반응
- **호감도**: 사용할수록 관계가 깊어져요 (내 정보에서 확인)
- **30초 방치**: 잠듭니다 (마우스 움직이면 깨어남)

## 테마

헤더 우측의 🌙/☀️ 버튼으로 다크/라이트 모드 전환.

## 도전과제 & 뱃지

도구 사용 시 자동 해금. **내 정보**(👤)에서 확인.

## AI 기능

1. [Google AI Studio](https://aistudio.google.com/apikey)에서 API 키 발급
2. 내 정보 → 설정 → Gemini API 키 입력
3. 저장 후 AI 도구 사용 가능

> API 키는 브라우저 localStorage에만 저장돼요.

개발자용으로 모델 ID·AI Studio/Vertex URL 등을 레포 한곳에서 맞추는 **`karmolab-ai`** 패키지가 있습니다. Toolbox **문서 → KarmoLabAI** 탭에서 쓰는 법을 정리해 두었어요.

## 데이터

모든 데이터(설정, 진행도, 도전과제)는 **브라우저 localStorage**에 저장돼요.

- 브라우저 데이터 삭제 시 초기화
- 다른 기기와 동기화 안 됨
- 내 정보 → 설정에서 수동 초기화 가능
