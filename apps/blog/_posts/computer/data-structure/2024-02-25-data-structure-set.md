## 정의

---

순서 없이 자료만 저장
집합 set  
다중집합 multi-set  

집합에 저장된 라료를 키 key  
집합은 동일한 키를 중복해서 가질 수 없음  

STL set은 템플릿을 사용해서 집합을 정의한 것  
집합에서는 여러 종류의 객체가 포함될 수 있다
