# YawnBot

게임/슬래시 커맨드 Discord 봇. 소스·데이터·이미지는 이 디렉터리가 기준입니다.

## 빠른 시작

워크스페이스 루트에서:

```bash
cd apps/discord-bots
npm install
npm run build:yawnbot
```

`.env`는 `apps/discord-bots/apps/yawnbot/.env.template`을 복사해 같은 폴더에 `.env`로 만듭니다.

```bash
npm run start:yawnbot
npm run deploy:yawnbot
```

전체 워크스페이스 설명은 `apps/discord-bots/README.md`를 참고하세요.
