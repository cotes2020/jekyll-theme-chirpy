# 블로그 포스트 작성법

## 1. 파일 만들기

`_posts/` 폴더에 아래 형식으로 파일 생성:

```
YYYY-MM-DD-제목.md
```

예시: `2026-03-20-내가-배운것.md`

---

## 2. Front Matter (필수)

파일 맨 위에 반드시 작성:

```yaml
---
title: 포스트 제목
date: 2026-03-20 10:00:00 +0900
categories: [대분류, 소분류]
tags: [태그1, 태그2, 태그3]
---

여기부터 본문 작성...
```

- **categories** — 최대 2단계, 사이드바 카테고리에 표시됨
- **tags** — 개수 제한 없음, 소문자 권장

---

## 3. GitHub에 올리기

```bash
cd ~/Desktop/LEo-dev5.github.io

git add _posts/2026-03-20-내가-배운것.md
git commit -m "add: 새 포스트"
git push
```

몇 분 후 https://leo-dev5.github.io 에 자동 반영.

---

## 4. 추가 옵션

```yaml
pin: true       # 상단 고정
toc: false      # 목차 숨기기
image:
  path: /assets/img/썸네일.png
```

---

## 5. 로컬 미리보기 (선택)

```bash
bundle exec jekyll serve
# http://localhost:4000 에서 확인
```
